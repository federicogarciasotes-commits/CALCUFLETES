Después de un cambio significativo en el proyecto, usar los siguientes comandos para guardar una versión en git:
git add .
git commit -m "Descripción clara del cambio"

Para empezar a trabajar, usar el siguiente comando:
venv\Scripts\activate

Para levantar el back:
uvicorn app.main:app --host 0.0.0.0 --port 8000 --ssl-keyfile certs/dev-key.pem --ssl-certfile certs/dev-cert.pem
Para levantar el front:
npm run dev
