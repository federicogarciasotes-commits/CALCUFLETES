1. Tener instalado Python (py --version)
2. Usar los siguientes comandos:
	python -m venv venv
	venv\Scripts\activate
	pip install -r requirements.txt
3. Instalar Node.js: nodejs.org
4. Usar estos comandos:
	npm install
	npm run dev

5. Crear certificado para poder usar local:
winget install FiloSottile.mkcert
mkcert -install <- cerrar la terminal y volverla a abrir
mkdir certs
mkcert -key-file certs/dev-key.pem -cert-file certs/dev-cert.pem [localhost] [otras IP que quieras]
